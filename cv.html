WESTBEN PERFORMER-COMPOSER RESIDENCY // Interactive Web Designer, AI, Musician, Storyteller
July 2021

project name: remember me otherwise
remember me otherwise is an interactive web installation based on body memory. In this installation, layers of video and audio recall childhood memories, each of which is encapsulated by a single hand gesture. As audiences interact with the videos, a pose estimation AI program tracks their own hand movements via the computer webcam, allowing them to use similar gestures to control the playback and mix. This project used the open sourced Tensorflow.js hand pose detection model to track the audience's hand gestures. This interactive website consists of 6 videos layered on top of each other and 6 recognizable hand gestures trained for this project with the model. The application is currently deployed on Heroku.

CALARTS EXPO // Creative Coder, visual artist, producer, cellist, post-production, Interactive Web Designer
May 2021

project name: WORDS TRIGGER
This project connects RunwayML, Processing, MaxMsp and TouchDesigner to create an audio and visual rich performance. The audio and visuals are all triggered by the text typed into the program. Depending on the number of words typed in, Processing will grab the same amount of colour values from random points of the collaged image created by RunwayML's AttnGAN. These colour values will be converted first into MIDI notes then turned into a readable score the Cellist can improvise off from. MaxMsp is programmed to pitch detect 36 notes ranging from C2-B4. As the Cellist plays, her notes will cause the TouchDesigner particle system to change shapes, colour and positions.

{ RUNWAYML | PROCESSING | MAXMSP | TOUCHDESIGNER | CELLO | PROJECTION | HTML }

SIMPLE STEP // Lead Developer
April 2021 - May 2021

Simple Step is, as it's name suggests, is a simple and easy to understand step sequencer. For those who do not know what a step sequencer is, it is a interface that allow you to set a number of pitches and rhythm as "steps" in the constantly looping machine. It was designed for absolute beginners having in mind they may not know musical terms and skills. Simple Step not only allows the user to easily play and create music but also teaches them about basic musical components.
